## 一个函数如何返回多个值？

1. 引用传参
2. 使用`tuple`
3. 使用`pair<T,bool>` 来判断函数可能会运行失败的情况
4. 使用

```cpp
#include <iostream>
#include <optional>

using namespace std;

struct Out {
    string out1 { "" };
    string out2 { "" };
};

optional<Out> func(const string& in) {
    Out o;
    if (in.size() == 0)
        return nullopt;
    o.out1 = "hello";
    o.out2 = "world";
    return { o };
}

int main() {
    if (auto ret = func("hi"); ret.has_value()) {
        cout << ret->out1 << endl;
        cout << ret->out2 << endl;
    }
    return 0;
}
```

std::nullopt 是 C++ 17 中提供的没有值的 optional 的表达形式，等同于 { } 。
